# GitHub Actions Demo - Shopping Cart

A simple Node.js shopping cart utility used to demonstrate GitHub Actions CI/CD workflows.

## Functions

- **addItem(cart, item, price)** — Adds an item to the cart
- **removeItem(cart, name)** — Removes an item from the cart by name
- **getTotal(cart)** — Returns the total price of all items in the cart

## Demo UI

Open `index.html` in a browser for an interactive visual demo of the cart functions.

## Scripts

```bash
npm test    # Run Jest tests
npm run lint # Run ESLint
```
